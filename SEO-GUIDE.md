## 🥇 Phase 1 — Get indexed (do FIRST — biggest, fastest win, ~30 min)
The "Welcome to WordPress" result in Google is stale cache from the old site. Force a recrawl:

1. **Google Search Console** → https://search.google.com/search-console
   - Add property: `themahirtech.com` (Domain property)
   - Verify with the **DNS TXT record** they give you (add it where your domain DNS lives)
2. **Submit sitemap:** Sitemaps → enter `sitemap.xml` → Submit
3. **Request indexing:** URL Inspection → paste each URL → "Request Indexing":
   - `https://themahirtech.com/`
   - `/about`, `/services`, `/projects`, `/industries`, `/contact`
   - each service page (`/services/ai-solutions`, `/services/ai-automation`, …)
4. **Bing Webmaster Tools** → https://www.bing.com/webmasters → add site, import from GSC, submit sitemap.
   (Bing also powers ChatGPT search — important for "AI search" visibility.)

➡️ Brand searches ("the mahir tech", "mahirtech") should rank #1 within days of this.
